Drip Detective

Drip Detective is a Web Application that my friend Reid and I made during the 2019 Madhacks Carbon competition. The theme of this hackathon was sustainability, so we created a shower timer/tracker with HTML, JavaScript, and a Bootstrap framework. The goal of the timer was to calculate water usage, and store it along with log in information using Google Firebase. I developed the scripts as well as the front end interface, while Reid handled the back end data storage. This was a great learning experience for me because it was my first time using HTML and JavaScript.

Irish Poker

Irish Poker is a card game that I recreated using Java. This project was made summer 2019 as a way to teach myself in between semesters and I decided to improve it in Winter 2020. I used an ArrayList to dynamically store the deck of cards, and a random number generator to select one. The selected card is compared to user input and the game progresses accordingly.
